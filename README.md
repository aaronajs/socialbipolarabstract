# Social Bipolar Abstract Argumentation Theory
#### Author: Aaron Suman

Source code for 4th year Individual Project.
This application requires Python 3.8.1 or above to run.

### Build and Run Instructions

1. Open Terminal Prompt.
2. Navigate to source folder, `sourcecode`.
    1. Create debate in a .txt file in the `debates` folder.
    2. Or use an existing debate in the folder.
3. To execute the evaluation process, run: `python control.py <filename>`

For example: `python control.py d30`

The application process will be updated in the terminal interface, and a PDF file of the debate will be generated and opened upon completion.
